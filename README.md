# Contact Manager App

A simple React + Node.js contact management application that allows users to add, search, and delete contacts.

## 📦 Tech Stack

- Frontend: React
- Backend: Node.js
- Database: lowdb

## 🚀 Features

- Add new contacts (name, email)
- View and search existing contacts
- Delete contacts
- Clean, responsive UI

## 🛠️ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/SowmyaKoneti/contact-manager.git
cd contact-manager
cd backend
npm install
cd frontend
npm install

# Start backend (in one terminal)
cd server
npm run dev

# Start frontend (in another terminal)
cd client
npm run dev

visit: http://localhost:5173/
